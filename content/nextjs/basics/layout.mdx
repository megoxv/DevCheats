# Layouts in Next.js

Layouts allow you to create reusable structures for your pages.

## Basic Layout

Example:
```jsx
// components/Layout.js
export default function Layout({ children }) {
  return (
    <div>
      <header>My Header</header>
      <main>{children}</main>
      <footer>My Footer</footer>
    </div>
  );
}
