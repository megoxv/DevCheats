# Notifications in FilamentPHP 3

Notifications are used to provide feedback to users.

## Sending a Notification

Use the `Notification` class to send notifications.

```php
use Filament\Notifications\Notification;

Notification::make()
    ->title('Saved successfully')
    ->success()
    ->send();
